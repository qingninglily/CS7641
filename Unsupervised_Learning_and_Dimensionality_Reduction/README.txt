Supervised Learning Homework ReadMe

1. Download the data in the github: 

https://github.com/qingninglily/CS7641/tree/main/Unsupervised_Learning_and_Dimensionality_Reduction

Dataset -- Credit Approval: crx.data 

Note: The second dataset I use can be loaded by running Python jupyer notebook -- Unsupervised Learning and Dimensionality Reduction--BreastCancer.ipynb

2. Open the ipynb files using Jupyter Notebook

There are 2 files in the GitHub:

Unsupervised Learning and Dimensionality Reduction--Credit
Unsupervised Learning and Dimensionality Reduction--BreastCancer.ipynb

3.Install Packages
Use "Pip install [package name]" command to install the packages

4. Run code:
You can click "Kernel" >> "Restart & Run All" Or you can run the cell one by one.

5. Open the command line:
Use "cd [Path] to run_experiments.py" to the location where you save the run_experiments.py file

6.In the command line, please type "python run_experiments.py" to run the code. Then you will see the images saved in the "image" folder and model results table printed in the command line.
