Reinforcement Learning Homework ReadMe

1. Download the code in the github: 

https://github.com/qingninglily/CS7641/Reinforcement_Learning

2.Install Packages
Use "Pip install [package name]" command to install the packages

3. Open the command line:
Use "cd [Path]" to run_experiments.py to the location where you save the run_experiments.py file

4.In the command line, please type "python run_experiment.py --all" to run the code. 

5.In the command line, please type "python run_experiment.py --plot" to run the code. Then you will see the images saved in the "report" folder
