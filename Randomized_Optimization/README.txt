Randomized Optimization Homework ReadMe

1. Download the data in the github

Dataset -- Breast Cancer: Breast_cancer_data.csv

2. Open the ipynb files using Jupyter Notebook

There are 3 files: 
Randomized Optimization -- 8 Queens.ipynb
Randomized Optimization -- Max K Color.ipynb
Randomized Optimization -- Travelling Salesperson.ipynb

3.Install Packages
Use "Pip install [package name]" command to install the package

4. Run code:
You can click "Kernel" >> "Restart & Run All" Or you can run the cell one by one.
